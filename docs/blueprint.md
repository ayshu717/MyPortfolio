# **App Name**: AD Portfolio

## Core Features:

- Sticky Header: A sticky header that becomes slightly transparent and blurred when scrolling. Includes a logo, navigation links, and a theme switcher.
- Hero Section: Displays a professional headshot, name, title, bio, action buttons, and social links with subtle 'fade in and slide up' animations.
- Skills Showcase: Organizes skills into categories (Top Skills, Languages, Databases, Tools & Soft Skills) using cards and icons.
- Project Display: Presents featured projects in a grid of cards, with an image or auto-playing carousel, title, description, tech badges, and links to source code and live demo.
- Co-curricular Activities: Cards displaying co-curricular activities. For 'Flora Flicks' utilize auto-playing carousel to display images. Links to Instagram.
- Contact Form: A 'Get in Touch' form within a card. Form Submission will trigger sending data to a Formspree endpoint with success/error message toasts.
- Comprehensive Footer: A multi-column footer with sections for 'Latest Projects,' 'Quick Links,' and 'Contact Information,' including a copyright notice and social media links.

## Style Guidelines:

- Light Mode: White (#F5F5F5) or light gray for the background.
- Dark Mode: Very dark gray (#121212) or black for the background.
- Primary accent color: Purple/violet (#A663CC, HSL 269 83% 70%).
- Font: 'Manrope' (sans-serif) for all text. Note: currently only Google Fonts are supported.
- Use relevant icons from libraries like FontAwesome or Feather Icons for skills, categories, and social links.
- Clean, single-page layout with a sticky header and detailed footer.
- Subtle 'fade in and slide up' animations on Hero Section elements using Framer Motion.
